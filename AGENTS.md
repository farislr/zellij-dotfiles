## OVERVIEW

Zellij user configuration with OS-specific keybindings. Installs zellij config + binary + terminal config (foot for Linux, alacritty for macOS).

## STRUCTURE

```
.config/zellij/
├── config.shared.kdl    # Plugins, load_plugins, web_client, global options
├── config.Linux.kdl     # Linux keybindings (secondary_modifier "Alt")
├── config.Darwin.kdl    # macOS keybindings (secondary_modifier "Shift+Cmd")
├── install.sh           # Unified installer (zellij + terminal config)
├── foot/                # Foot terminal config (Linux)
│   ├── foot.ini
│   └── dank-colors.ini
├── alacritty/           # Alacritty terminal config (macOS)
│   └── alacritty.toml
└── tests/               # bats test suite
    ├── terminal-install.bats
    └── test_helper.bash
```

## INSTALLATION

```bash
# Full install: zellij config + binary + terminal config
./install.sh

# Terminal config only
./install.sh --terminal-only

# Skip terminal config
./install.sh --no-terminal

# Zellij config only (skip binary + terminal)
./install.sh --config-only
```

## CONVENTIONS

- **Config split**: Shared in `config.shared.kdl`, OS-specific in `config.Linux.kdl` / `config.Darwin.kdl`
- **Install script**: Auto-detects OS, creates timestamped backups, symlinks zellij config
- **Terminal install**: Uses `--terminal-only` flag, copies configs to `~/.config/foot/` or `~/.config/alacritty/`
- **Do NOT edit**: `config.kdl` directly - it is a symlink generated by `install.sh`
